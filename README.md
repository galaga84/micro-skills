# Micro Skills

Base web construida con React 19, Vite 8, Tailwind CSS 4 y daisyUI 5.

## Requisitos

- Node.js 20.19 o superior
- npm 10 o superior

## Desarrollo

```bash
npm install
npm run dev
```

## Comprobaciones

```bash
npm run check
```

`check` ejecuta el análisis estático y genera la compilación de producción.

## Otros comandos

- `npm run lint`: revisa el código con Oxlint.
- `npm run build`: genera la aplicación en `dist/`.
- `npm run preview`: sirve localmente la compilación de producción.
